\doxysubsection{Detailed Description}
4a -\/ one parameter set (with additional information)

4 -\/ In case of identifiability or sensitivity, point modifications of a parameter set have to be done (i.\+e. changing one parameter value only.) Here is a data structure to store the cost of varying a parameter around an initial parameter set. It can be used during sensitivity analysis (the cost of just changing one parameter around the original one), or during identifiability analysis (storing the best set after optimization if this parameter is fixed to this value) It can also stores the statistics of the cost for each variable.
