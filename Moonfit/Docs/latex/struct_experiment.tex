An experiment is the design of a simulation performed on a model. The same experiment can be performed on different models. An experiment can have several conditions. The \mbox{\hyperlink{struct_experiment}{Experiment}} class is used as mother class for each user-\/defined experiment, and implements all basic functions for simulations and data management.
