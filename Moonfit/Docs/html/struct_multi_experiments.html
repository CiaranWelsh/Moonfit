Detailed Description

A MultiExperiments is made from multiple used-defined Experiments, and allows to simulate them together and get a multi-experiment cost between simulations and data. The same model is used for each experiment, therefore the same model parameters. Each experiment can be given a coefficient for the total cost calculation.
