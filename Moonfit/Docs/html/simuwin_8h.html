◆ ALLOW_OPTIMIZE

#define ALLOW_OPTIMIZE

This is the main interface to perform complex tasks with a model/experiment. The job is separated into two main classes :

  • manageSims, which is independent of any graphical library. it can perform simulations, optimizations, load configuration files, saving results as tables, etc.

  • simuWin, which is extending manageSims with a graphical QT interface. some actions are only possible with simuWin, such as directly saving simulations as plots.

    As Qt doesn't support plotting graphs, two additional libraries are proposed(one is enough)

    • Qwt (older one, incompatible between different versions --> often raises compiling errors)
    • QCustomPlot (more recent, makes beautiful plots, a bit slower, might cause trouble if plots a whole optimization)
