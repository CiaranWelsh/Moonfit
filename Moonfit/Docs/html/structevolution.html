Detailed Description

Tool data structure to track the evolution of a variable with time, such as the cost during the optimization. In order to avoid reallocating a bigger table every time, it records by blocs (of size 'SizeGroups'), and everytime a block is finished, computes min, max, mean and stddev of this block and starts a new block.
