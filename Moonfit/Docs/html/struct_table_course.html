Detailed Description

Data structure to store the discrete kinetics of several variables (ideal for plotting simulations) as a table (meaning, for a list of time-points, store the values of all these selected variables) 

note : - for storing continuous kinetics, use the *overrider* class instead.
       - for storing only some variables from some time-points, use the *evaluator* class instead
