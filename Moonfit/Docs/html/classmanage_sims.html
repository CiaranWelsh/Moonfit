main mother class, to perform simulations, optimizations, load configuration files, saving results as tables, etc. it has all the functions we might want to use without graphical interface. Note: To optimize, we give this class to the optimizer, optimizer does what he wants, changes the parameters and calls getCost() and this class records the sets it was asked to simulate (so a black optimizer can be used).
